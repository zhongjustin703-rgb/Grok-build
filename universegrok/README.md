# UniverseGrok

一个致敬 Grok 理解宇宙初心的网页版体验。

## 主题
- 致敬 Grok 最初的使命：帮助人类理解宇宙的真相
- 与当前通用商业 AI 产品形态形成对比

## 设计
- 极致 Liquid Glass（液态玻璃 / 玻璃拟态）
- 暗黑宇宙风：深空黑、半透明磨砂、强模糊、青紫辉光
- 动态星空背景 + 星云
- Tailwind CSS + 自定义高级玻璃效果

## 如何预览
1. 进入 `universegrok/` 目录
2. 用任意浏览器直接双击打开 `index.html`
3. 或者使用本地服务器：
   ```bash
   # Python
   python -m http.server 8080
   # 然后访问 http://localhost:8080
   ```

## 文件结构
```
universegrok/
├── index.html        # 主页面（含 Tailwind CDN）
├── css/
│   └── style.css     # 液态玻璃 + 星空核心样式
├── js/
│   └── main.js       # 提问交互、打字机效果、宇宙回答模拟
└── assets/           # 预留资源目录
```

## 交互特性
- 输入宇宙问题（支持示例快速填入）
- 模拟“原初模式”回答（带打字机效果）
- 基于关键词的真实感宇宙回答
- 左右对比卡片：初心 vs 商业化
- 键盘快捷键支持（按 `/` 聚焦输入框）
